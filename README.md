# 🏦 Loan Management System

## 📌 Project Overview
This is a Full Stack Loan Management System developed using **React (Frontend)** and **FastAPI (Backend)**.  
The system allows users to check loan eligibility, calculate EMI, and simulate real-world loan processing.

---

## 🚀 Features
- ✅ Loan Eligibility Check  
- 💰 EMI Calculation  
- 📊 Credit Score Validation (Optional)  
- 📅 Application Date Handling  
- 🏦 Loan Type Selection  
- ⚡ Real-time API Integration  

---

## 🛠 Tech Stack
### Frontend:
- React (Vite)
- JavaScript
- Axios

### Backend:
- Python
- FastAPI
