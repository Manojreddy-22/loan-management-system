# Loan Management System

## 🚀 Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI
- API: REST

## 📌 Features
- Loan eligibility check
- EMI calculation
- Credit score validation
- Real-time API integration

## ▶️ Run Backend
cd backend
python -m uvicorn main:app --reload

## ▶️ Run Frontend
cd frontend
npm install
npm run dev
